# Database and SQL

사용 DBMS
- MariaDB
- OracleDB

이전에 Bitnami Server를 설치할 당시, Apache Web Server와 MariaDB를 설치하면서 Mysql의 환경도 같이 조성되었기 때문에 따로 Mysql을 중복으로 설치하지 않았다.
(설치시 port number를 따로 지정하면 Mysql과 MariaDB 환경을 동시에 조성할 수 있으나 두 프로그램 간에 충돌 발생 우려로 MariaDB로 진행함)
